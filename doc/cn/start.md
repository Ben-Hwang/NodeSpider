# Feature

- 开箱即用，用最少的代码快速开发五脏俱全的爬虫程序
- 计划规则与任务相分离，再复杂的爬取需求也可以轻松实现
- 简单好用的数据管道，保存抓取数据是一件轻松的事情
- 自动转码 utf8、jQ选择器……可爱的小功能该有的都有
- 性能优异，你对异步并发有绝对的控制自由
- 丰富、简约的拓展接口，支持你天马行空的想象力和伟大需求

# Install

npm:

```
npm install nodespider --save
```

yarn:

```
yarn add nodespider
```
